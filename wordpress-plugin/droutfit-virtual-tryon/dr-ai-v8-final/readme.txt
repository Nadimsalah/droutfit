Direct AI Virtual Try-On for WooCommerce. Enable customers to visualize garments on themselves using deep-learning technology.

== Description ==

DrOutfit Virtual Try-On provides an interactive AI experience for WooCommerce stores. This plugin adds a "Virtual Try-On" button to product pages, allowing users to upload a photo and see an AI-generated preview of themselves wearing the selected item.

The core technology uses deep-learning models to process body shapes and textures, aiming to provide a realistic visualization. This helps customers make more informed purchasing decisions by seeing how a garment might look in a real-world scenario.

== Installation ==

1. Upload the `droutfit-virtual-try-on` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the DrOutfit Settings menu in your WordPress dashboard.
4. Log in with your DrOutfit account to connect your store.

== Frequently Asked Questions ==

= Does it work with any theme? =
Yes, the plugin uses standard WooCommerce hooks and is designed to be compatible with most WordPress themes.

= Is it mobile-friendly? =
Yes, the AI Try-On interface is fully responsive and optimized for mobile browsers.

= Which products support Try-On? =
You can enable or disable the Try-On feature for individual products directly from the WooCommerce product editor.
